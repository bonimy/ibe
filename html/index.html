IRSA Image Server

The IRSA Image Server provides low-level, program friendly methods for querying astronomical image metadata tables, including support for the IVOA Simple Image Access protocol. Images and image cutouts are available for direct download (e.g. via wget or curl), and are laid out in browsable directories.

Image metadata tables may be queried by appending a query string to a base URL identifying the table. For SIA queries, these base URLs have the following structure:

<server><root>/sia/{mission}/{data-set}/{table-name}

For queries returning data in simpler formats (IPAC ASCII, CSV, TSV), the base URLs look like:

<server><root>/search/{mission}/{data-set}/{table-name}

Data products associated with the table (mostly images) are URL accessible from:

<server><root>/data/{mission}/{data-set}/{table-name}

Finally, the URL structure of these data products is documented by:

<server><root>/docs/{mission}/{data-set}/{table-name}
